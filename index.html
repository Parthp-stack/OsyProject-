Algorithm Info

Algorithm Type Explanation / Downside
FCFS Non-Preemptive Long waiting time for short processes (convoy effect)
SJF Non-Preemptive Can cause starvation for longer processes
SRTF Preemptive High context switching overhead; shorter processes get priority
Round Robin (RR) Preemptive Time quantum choice affects performance; too small → many switches, too large → behaves like FCFS
Priority Non-Preemptive / Preemptive Can cause starvation for low-priority processes; preemption optional
